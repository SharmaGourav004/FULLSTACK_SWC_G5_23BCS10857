# Day 8 Homework

This workspace contains three separate Vite + React apps:

1. `Q27` - shared global cart state with add, remove, quantity updates, and total calculation.
2. `Q28` - asynchronous content loading with Redux Thunk and loading/success/error states.
3. `Q29` - dynamic product filtering with category, price, rating, multi-filter support, and persisted filter state.

The cart app now splits the product and cart views into separate components under `Q27/src/components`.

## Run Any App

From inside the app folder:

```bash
npm install
npm run dev
```

To verify a production build:

```bash
npm run build
```
