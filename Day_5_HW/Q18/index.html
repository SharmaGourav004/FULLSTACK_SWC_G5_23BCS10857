Q18 - Stopwatch utility

Track elapsed time with a dashboard stopwatch

This stopwatch shows elapsed time clearly, supports start, pause, and reset actions, and keeps the layout simple enough for productivity-style time tracking.

00:00
Ready
Elapsed 0 milliseconds
Control state Idle
StartRuns the timer if it is not already active.
PauseStops the timer and keeps the current elapsed value.
ResetClears the timer back to zero and returns to ready mode.

The display uses HH:MM:SS once the timer reaches an hour; below that it shows MM:SS for a cleaner dashboard look.
